How to do the control charting?

The univariate control chart

Univariate control chart is the chart that monitors only one characteristic of a product. Bascially, all the charts are univariate intially.

Some univariate charts are Shewhart X-bar chart, CUSUM chart, EWMA chart, GWMA chart, HWMA chart.

Go check Multivariate Control Chart.

The multivariate control chart

The multivariate control chart monitors multiple characteriatics together. The scheme has been more complicated, but also more efficient, to some extent.

Some multivariate charts are Hoteling's T2 chart, HEWMA chart, MHWMA chart.

Go check Univariate Control Chart.

Others

There are improvements based on control charting methods for specific situations.

Nonparamenter

Nonparamenter Method for control charts are based on tests such like sign test, Wilcoson tests. The idea is to take such test statistics as the input of some control chart, e.g. CUSUM, EWMA, HWMA.

VSI

The VSI(variable sampling interval) scheme is to take different sampling intervals when charting. This is for the in-situ monitoring of the charting.

VSS

The VSS(variable sampling size) scheme is to take different sample size for each subgroup, such that to save the computing resource for the control chart.

VSSI

The VSSI(variable sampling and sample size) scheme. The scheme has two contents

  • VSI
  • VSS
For more details, see VSI and VSS.

CoDa-MHWMA

The CoDa-MHWMA control chart is the control chart that operates on the compositional data. For more details about the ARL performance, see Results.
